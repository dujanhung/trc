# TRC doc

# control chars

changes parser behaviors.

## separator

```txt
(O)
:&:
(...)
```

separates operators.

- (O) : an operator
- (...) : repeates

---

# operator

runs regex operations.

## subtitube

```txt
(S)
:=:
(R)
```

subtitubes texts.

- regex (S) : search pattern
- ref regex (R) : replacement

## erasure

```txt
(S)
```

erases texts.

- regex (S) : search pattern
