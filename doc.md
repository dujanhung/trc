# the API doc converter language

# <code>◆◇</code>

```txt
◆
input texts
◇
```

represents an abitrary text content.

# <code>▣</code>

```
▣
◆
metadata texts
◇
```

represents a metadata.

> [!NOTE]
> metadatas would be ignored at runtime.

# <code>▼▲</code>

```txt
▼
▣
◙
◘
▲
```

represents an entry.

# <code>▶◀</code>

```txt
▶◀
```

represents a variable.
